# Dotfiles

## Guide
- Configs
- Install
- Bin

### Configs 
```sh
SSH_EMAIL='YOUR EMAIL'
```

### Install
```sh
# Clone the repo dotfiles
$ git clone https://github.com/HallexCosta/dotfiles.git

# Access project cloned
$ cd dotfiles

# Run auto install
$ ./install.sh
```

### Bin
```sh
# Access the folder bin
$ cd bin
```
